pub fn read_strip(&mut self) -> TiffResult<DecodingResult>

👎Deprecated: Use read_chunk instead

Read a single strip from the image and return it as a Vector. This method does not return information on the size of the strip read and so the caller must take care of the final strip in the image as the data read may be less than the expected strip size. This interface may change at a future date to reflect this.

source

pub fn read_tile(&mut self) -> TiffResult<DecodingResult>

👎Deprecated: Use read_chunk instead

Read a single tile from the image and return it as a Vector. This method does not return information on the size of the tile read and so the caller must take care of tiles on the rightmost column and the bottommost row in the image as the data read may be less than the expected tile size. This interface may change at a future date to reflect this.
