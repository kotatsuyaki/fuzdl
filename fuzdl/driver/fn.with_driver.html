ยงReturn Value

The outer Result is of the Err variant if an error occurs in the driver creation or cleanup steps. The inner Option is of the None variant if a kill signal is received before the callback function completes. In both cases, the driver resources are properly cleaned up before returning.

ยงResource Leaks

In some cases, the resources may not be properly cleaned up. These cases include but are not limited to:

  • The callback function panics.
  • The driver creation functions panics.
